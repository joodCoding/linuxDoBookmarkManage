# linuxDoBookmarkManage

本脚本为二次开发，原作者地址：huanchong《L站收藏夹》你的 L 站必备油猴脚本（3 级福利）<https://linux.do/t/topic/1684359>。

> 当前版本没有在其他账号做过完整测试，可能存在 bug。

## 功能速览

- 收藏夹入口（右下角悬浮按钮）
- 手动同步服务器收藏数据（每次 20 条）
- 阅读视图（搜索、卡片缩放、色彩调节）
- 气泡模式（卡片自由拖拽、缩放，默认大小按楼层计算）
- 本地收藏与服务器收藏双向联动

## 脚本食用指南

1. 右下角收藏入口：点开即可使用。
2. 手动同步：点击“手动同步”按钮后，会请求服务器收藏数据（一次获取 20 条）。
3. 阅读视图：支持查找、缩放、色彩调整；切换到气泡模式后可拖拽与缩放卡片。
4. 建议使用方式：
   - 打开一个首页后，尽量通过中键滚轮在新标签页浏览帖子，减少整页刷新。
   - 在帖子页收藏时，使用右下角收藏按钮（会保存本地并调用服务器接口），否则本地可能不会及时同步更新。
